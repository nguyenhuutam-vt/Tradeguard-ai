# TradeGuard AI - Agent Instructions

## Project

TradeGuard AI is a Next.js + TypeScript crypto futures risk assistant.

It helps users:

- calculate risk before entering long/short futures trades
- estimate liquidation price
- analyze trade setups with AI
- keep a trading journal
- monitor watchlists and volatility

It must not:

- give financial advice
- guarantee predictions
- tell users to buy/sell/long/short immediately
- claim guaranteed profit or accuracy

## Required agent workflow

Before editing code:

1. Read the relevant files.
2. Understand the current architecture.
3. Write a short plan.
4. Identify risks and edge cases.
5. Make small changes.
6. Run checks if available.

After editing code:

1. Summarize what changed.
2. List files changed.
3. List tests/checks run.
4. Mention remaining risks.
